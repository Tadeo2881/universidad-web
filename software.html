Plan Académico de Software (3 Años)

Domina los lenguajes de programación más demandados y conviértete en un arquitecto de soluciones digitales escalables.

Semestre Asignaturas de Especialidad
1er Semestre Fundamentos de Programación, Lógica y Algoritmos
2do Semestre Desarrollo Web (HTML5, CSS3, JavaScript) y Bases de Datos
3er Semestre Programación Orientada a Objetos y Estructura de Datos
4to Semestre Desarrollo de Aplicaciones Móviles (iOS/Android) y Backend
5to Semestre Ingeniería de Software, Gestión de Proyectos y QA Testing
6to Semestre Inteligencia Artificial, Machine Learning y Proyecto Final
